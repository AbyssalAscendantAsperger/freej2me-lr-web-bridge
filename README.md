# freej2me-lr-web-bridge
FreeJ2ME-Plus Web Bridge - Prototype Node.js

Dự án prototype này giúp chạy freej2me-plus ở chế độ Libretro frontend (không mở cửa sổ AWT), giao tiếp qua stdin/stdout, rồi đưa hình ảnh và điều khiển lên web qua WebSocket.

Người dùng có thể upload file JAR/JAD từ trình duyệt, server sẽ tự động khởi động/restart emulator và chạy game.
