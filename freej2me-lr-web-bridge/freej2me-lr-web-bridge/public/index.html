Hướng dẫn

1. Upload game: Chọn file JAR/JAD từ máy tính. Server sẽ tự động khởi động FreeJ2ME và chạy game.

2. Bàn phím:
↑ ↓ ← → : Điều hướng
Enter : Fire
F1 : Left Soft Key
F2 : Right Soft Key
0–9, *, # : Phím số
Esc : Clear

3. Cảm ứng / Chuột: Click hoặc chạm vào màn hình để gửi sự kiện touch.
